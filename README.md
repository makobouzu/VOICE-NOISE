# VOICE-NOISE
"VOICE | NOISE" is a sound mapping and recording tool using RNNoise, Recorder.js, and mapbox.  
It works in Safari.

## Install
```sh:fish
git clone git@github.com:makobouzu/VOICE-NOISE.git
cd VoiceNoise
npm install
```

## Demo
You need to create a .env file and database with a postgresql.  

```sh:fish
cd VoiceNoise
npm run devStart 
```
